# Automated-Portfolio-Analytics-Risk-Assessment

Features
✅ Stock Data Retrieval – Fetches historical stock prices using yfinance.
✅ Portfolio Returns Calculation – Computes log returns and cumulative performance.
✅ Risk Assessment – Measures Sharpe ratio, volatility, and max drawdown.
✅ Interactive Visualization – Plots cumulative returns and drawdowns with Matplotlib.
✅ Automated Reporting – Exports performance metrics to an Excel report.

Technologies Used
Python – Primary programming language
yfinance – Stock market data retrieval
pandas, numpy – Data manipulation and analytics
Matplotlib, Plotly – Data visualization
Excel (pandas) – Automated reporting
